# 选择器

## 基本结构

```html
<select class="j-select" id="" name="" autocomplete="">
    <option value="">请选择</option>
    <option value="1">中国</option>
    <option value="2">美国</option>
    <option value="3">英国</option>
</select>
```

## 应用示例

<div>
    <select class="j-select">
        <option>请选择</option>
        <option value="1">中国</option>
        <option value="2">美国</option>
        <option value="3">英国</option>
    </select>
</div>

## 样式

1. 仅提供 `select` 的基础样式。
2. 在 `form` 中，针对不同方向的表单，`select` 会有不同宽度的适应样式。

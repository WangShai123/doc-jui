# CSS 工具类

`JUI` 提供和 `tailwind css` 一致的常用 `css` 工具类。

本文档不提供具体工具类的说明，具体规则请查看 [`tailwind css`](https://tailwindcss.com/docs/display) 文档。

其余工具类包含：

- `flex` Flex CSS
- `grid` Grid CSS
- `sizing` 尺寸
- `display` 显示类型
- `float` 浮动
- `object-fit` 内容适应
- `object-position` 对象位置
- `position` 定位
- `margin` 外边距
- `padding` 内边距
- `overflow` 溢出
- `visibility` 可见性
- `z-index` 层叠顺序
- `font-size` 字号
- `font-style` 字体样式
- `font-weight` 文字加粗
- `letter-spacing` 文字间距
- `line-height` 行高
- `list-style` 列表标记样式
- `text-align` 文本对齐
- `text-decoration` 文本装饰线
- `text-transform` 文本转换
- `text-overflow` 文本溢出
- `text-indent` 文本缩进
- `vertical-align` 垂直对齐
- `white-space` 空白符
- `word-break` 字符换行
- `content` 字符插入
- `background` 背景
- `border` 边框
- `outline` 轮廓
- `box-shadow` 盒阴影
- `text-shadow` 文本阴影
- `opacity` 透明度
- `blur` 模糊滤镜
- `drop-shadow` 投影滤镜
- `transitions` 过渡效果

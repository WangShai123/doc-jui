```js
 */

```

## 获取当前语言

`getLocale()`

- 优先读取 `html` 标签的 `lang` 属性字符串的前两位以判断（`en`、`zh`），否则回退到 `getLang()`。

`getLang()`

- 通过 `navigator.language` 判断是否以 `en` 开头来返回 `en` 或 `zh`；
- 在无 `navigator` 环境下返回 `en`。

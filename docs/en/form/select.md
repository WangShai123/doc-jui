# Select

## Basic Structure

```html
<select class="j-select" id="" name="" autocomplete="">
    <option value="">Please select</option>
    <option value="1">China</option>
    <option value="2">USA</option>
    <option value="3">UK</option>
</select>
```

## Examples

<div>
    <select class="j-select">
        <option>Please select</option>
        <option value="1">China</option>
        <option value="2">USA</option>
        <option value="3">UK</option>
    </select>
</div>

## Styles

1. Only provides basic styles for `select`.
2. In `form`, `select` will have different width adaptation styles for forms in different directions.

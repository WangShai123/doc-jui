# CSS Utilities

`JUI` provides common `css` utility classes consistent with `tailwind css`.

This document does not provide specific utility class descriptions. Please refer to the [`tailwind css`](https://tailwindcss.com/docs/display) documentation for specific rules.

Other utility classes include:

- `flex` Flex CSS
- `grid` Grid CSS
- `sizing` Size
- `display` Display type
- `float` Float
- `object-fit` Content fit
- `object-position` Object position
- `position` Position
- `margin` Margin
- `padding` Padding
- `overflow` Overflow
- `visibility` Visibility
- `z-index` Stacking order
- `font-size` Font size
- `font-style` Font style
- `font-weight` Font weight
- `letter-spacing` Letter spacing
- `line-height` Line height
- `list-style` List style
- `text-align` Text alignment
- `text-decoration` Text decoration
- `text-transform` Text transform
- `text-overflow` Text overflow
- `text-indent` Text indent
- `vertical-align` Vertical alignment
- `white-space` White space
- `word-break` Word break
- `content` Content insertion
- `background` Background
- `border` Border
- `outline` Outline
- `box-shadow` Box shadow
- `text-shadow` Text shadow
- `opacity` Opacity
- `blur` Blur filter
- `drop-shadow` Drop shadow filter
- `transitions` Transition effects

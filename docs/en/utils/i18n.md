## Get Current Language

`getLocale()`

- Priority read first two characters of `html` tag's `lang` attribute to determine (`en`, `zh`), otherwise fallback to `getLang()`.

`getLang()`

- Return `en` or `zh` by checking if `navigator.language` starts with `en`;
- Return `en` in environments without `navigator`.

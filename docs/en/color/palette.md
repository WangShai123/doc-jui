# Color System

JUI provides a gorgeous, accessible color system for designing beautiful and accessible websites and applications.

1. **Automatic Dark Mode**: Each color automatically switches variants based on the theme mode.
2. **Transparent Variants**: Each color has a transparent variant for creating semi-transparent elements like buttons and cards, which is also very useful for creating blends and gradients.
3. **P3 Gamut Support**: Considering blending differences in wide-gamut color spaces, `P3 gamut` is supported by default, ensuring better color performance on advanced user devices.
4. **APCA Text Contrast**: Uses modern `APCA` algorithms to ensure text readability on different background colors, aligning better with human perception.
5. **Accessibility**: Guarantees sufficient and reasonable contrast between text color and corresponding background color. Compliant with `WCAG 2.1 AA` standards, ensuring good readability even under low contrast.
6. **Color Pairing**: Click the color blocks below to view suggested color pairings.
